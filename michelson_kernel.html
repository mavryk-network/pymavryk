michelson-kernel

michelson-kernel [OPTIONS] COMMAND [ARGS]...

Options

--version

Show the version and exit.

install

Install Michelson kernel in current environment

michelson-kernel install [OPTIONS]

remove

Remove Michelson kernel from current environment

michelson-kernel remove [OPTIONS]

run

Run Michelson kernel

michelson-kernel run [OPTIONS] [ARGS]...
